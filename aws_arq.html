LUMINAI Refinery Advisor en AWS

El LUMINAI Refinery Advisor es un sistema de soporte a la decisión impulsado por inteligencia artificial, diseñado para optimizar la eficiencia operativa en refinerías. Este sistema proporciona recomendaciones en tiempo real para ayudar a los operadores a tomar decisiones informadas. La arquitectura de referencia que exploraremos ilustra un despliegue altamente escalable de LUMINAI en AWS, utilizando varios servicios gestionados de AWS para recopilar, procesar y almacenar datos operativos de múltiples refinerías.

Servicios clave de AWS utilizados

  • Amazon ECS: Utilizado para la ejecución de contenedores que procesan datos en tiempo real y gestionan los servicios de negocio.
  • Amazon Aurora: Base de datos relacional altamente escalable y eficiente para el almacenamiento de resultados de procesamiento.
  • Amazon MSK: Facilita la ingesta y procesamiento de datos en streaming.
  • Amazon MQ: Servicio de mensajería para la gestión de datos entre servicios.

Descripción de la arquitectura punto por punto

  1. Ingesta de datos: Los datos operativos de las refinerías se transmiten al LUMINAI Refinery Advisor en AWS a través de un Application Load Balancer (ALB) en una subred pública, dirigiendo el flujo de datos a un pipeline de procesamiento de sensores en una subred privada.
  2. Procesamiento de datos en tiempo real: Los procesos de ingesta de datos en streaming ejecutados en Amazon ECS dirigen los flujos a Amazon Managed Streaming for Apache Kafka (Amazon MSK), que ingesta y procesa datos en tiempo real a gran escala.
  3. ETL y almacenamiento: Los resultados de cada trabajo de transformación y carga (ETL) se almacenan en Amazon Aurora PostgreSQL y se envían a Amazon MQ, que es un servicio de mensajería altamente eficiente y escalable.
  4. Servicios empresariales: Los servicios de orquestador y razonador de LUMINAI se ejecutan en Amazon ECS y procesan mensajes a través de Amazon MQ, aprovechando algoritmos impulsados por IA para proporcionar información en tiempo real.
  5. Acceso a datos: Los datos procesados son almacenados en Aurora PostgreSQL y están disponibles para herramientas de interfaz web, API y BI, mientras que los datos a largo plazo se guardan en Amazon Simple Storage Service (Amazon S3).
  6. Interfaz de usuario: Los usuarios, incluyendo gerentes de planta, equipos de mantenimiento y científicos de datos, acceden a los datos a través de una interfaz web o APIs, obteniendo una visión integral de cada refinería.
